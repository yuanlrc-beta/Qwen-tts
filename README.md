# Qwen-tts
千问tts一键安装整合包，可视化图形界面的千问TTS模型语音合成克隆工具
1、可以自定义音色描述，然后输入文字生成相应的音频。
2、可以上传自定义音频克隆声音，支持音色保存，方便后续服用。
3、支持CPU和GPU两种模式。
4、支持自定义参数、自定义配置信息等。
#环境要求：
python3.10及其以上
PyQt6>=6.4.0
soundfile>=0.12.0
sounddevice>=0.4.6
qwen_tts
openai-whisper
torch>=2.0.0
numpy>=1.21.0
支持50系显卡
